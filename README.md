# EcoRide

EcoRide is a PHP MVC web application for ecological carpooling.

## Features
- User authentication
- Ride creation & participation
- Ecological vehicle detection
- Ride history (driver & passenger)
- Reviews & ratings
- Admin & employee roles

## Tech Stack
- PHP (MVC architecture)
- MySQL
- Tailwind CSS (CDN)
- No framework (pure PHP)
